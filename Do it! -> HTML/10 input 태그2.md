## input 태그의 주요 속성
### 자동으로 입력 커서를 갖다 놓는 autofocus 속성 
*autofocus 속성을 사용하면 페이지를 불러오자마자 폼에서 원하는 요소에 마우스 포인터 표시 가능.
```html
기본형 -> <input type=텍스트-입력-필드 autofocus required>
```
