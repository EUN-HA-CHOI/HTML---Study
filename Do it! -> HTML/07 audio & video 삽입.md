## 오디오와 비디오 삽입하기  
### 다양한 멀티미디어 파일을 삽입할 때 쓰는 object , embed 태그  
``` html
기본형 -> <object width="너비" height="높이" data="파일"></object>

기본형 -> <embed src="파일 경로" width="너비" height="높이">   <!--<embed>태그는 닫는 태그가 없다.-->
```

* 웹 브라우저에서 지원하는 멀티미디어 파일의 종류 => 비디오는 mp4 ,오디오 파일은 mp3 형식을 주로 사용함.
