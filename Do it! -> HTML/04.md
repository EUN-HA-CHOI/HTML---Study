### 목록 만들기  
```html
순서 있는 목록을 만드는 <ol> , <li> 태그  

   기본형 -> <ol>
             <li>항목1</li>
             <li>항목2</li>
            </ol>
```  
##### *순서 있는 목록은 type 속성을 이용하여 순서를 나타낸다.  (start 속성을 사용하여 시작 번호를 바꿀 수 있다.)  
| 종류 | 설명 |
|----|----|
|type = "1" | 숫자 |
|type = "a" |영문 소문자|
|type = "i" |로마숫자 소|  

```html  
순서 없는 목록을 만드는 <ul> , <li> 태그  

  기본형 -> <ul>
            <li>항목1</li>
            <li>항목2</li>
          </ul>
```  

``` html
설명 목록을 만드는 <dl> , <dt> , <dd> 태그 

  기본형 -> <dl>
            <dt>이름</dt>
            <dd></dd>
          </dl>
