## 조건문  

* if (조건식)  
    실행문;  
    
```javascript
if (true) {
  console.log('Hello, if!')
//조건식이 true은 참 이므로 실행문이 실행되어 콘솔에 Hello, if!이 출력된다.

if (flase) {
  console.log('Hello, if!')
//조건식이 flase이면 거짓이므로 콘솔에 실행결과가 표시되지 않는다.  
}

=> flase, '',0,NaN,null,undefined 만 거짓인 값이고, 나머지는 참인 값이다 따라서  
   true로 형 변환되는 참인 값이나 그것을 담고 있는 변수를 넣어도 된다.
